Kategorie: Jak na to?
Týká se verze: Drupal 6.x

Ahoj potřeboval bych poradit ohledně odstranění položky menu pro anonymní uživatele. Napadlo mě, že bych umístil nějakou preproces funkci nebo theme funkci do souboru template.php kde by se zkontrolovalo jestli je uid == 0 a pokud ano tak udelat unset té menu položky… problém je, že nemůžu přijít na správný postup odchycení dané položky v menu a kterou preproces nebo theme funkci použít. Byl bych vděčný případně i za nějaký snippet…

Tak už mám hotovo. Problem je

Napsal uživatel Anonymous dne 31 Květen, 2013 - 19:07.
Tak už mám hotovo.
Problem je v tom, že přes ‚access callback‘ se dají řešit jen položky, které nejsou nastaveny jako výchozí tedy nemají „MENU_DEFAULT_LO­CAL_TASK“. V případě, že jsem chtěl právě skrýt výchozí položku tak se mi ukrylo celé Secondary tabs a ne jen ta požadovaná. Ale vyřešil jsem to přes themeName_pre­process_hook() kde dělám:
if (strpos($link,">Neco<") !== false) { // hide the 'Neco' tab
     return '';
}
a pomocnou funkci kde si zjišťuji stav zobrazit/nezo­brazit. A v případě, že by někdo chtěl natvrdo dát do prohlížeče adresu tak v ‚page callback‘ mám pomocnou funkci a zjišťuji jestli má oprávnění a pokud nemá vrátim mu:
drupal_access_denied();
return;
Samozřejmě ideální by bylo to řešit přes access callback a v 90% případů je to nejlepší nebo přes hook hook_menu_alter, ale tento hook se zase provede jen jednou a to při hook_menu() (tedy při registraci menu do systému a pak už ne :-( Díky všem za radu M.

Ještě menší přesnění, které

Napsal uživatel marinex dne 31 Květen, 2013 - 20:28.

Zapoměl jsem se přihlásit :-) ale uvedu ještě menší upřesnění, abych doplnil svoje předchozí řešení. Použil jsem YOURTHEME_menu_lo­cal_task (místo themeName_pre­process_hook()) včetně:

if (strpos($link,">Neco<") !== false) { // hide the 'Neco' tab
     return '<li class="unvisible" >' . $link . "</li>\n"; // pokud uz tam mate nejaku tridu pouzijte regex a pouze doplnte tu svoji
}

pak v css nastavim:

.unvisible {
    display: none;
}

A ještě jedna perlička, lze použít i jquery a najít si konkretní prvek a nastavit jí metodu .remove(), ovšem někdy se může stát že se javascript spustí později a pak tam ta položka problikne :-) Marinex
